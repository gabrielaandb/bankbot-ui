
# Bankbot UI

> UI for testing bankbot endpoints.

<img src="bankbot.jpg" alt="bankbot" style="width: 200px;"/>

## Usage

```bash
git clone ...
cd bankbot-ui/
open index.html
```

To point UI to a custom server, add 'url' query parameter to URL. Example:
`http://bankbotui.com?url=http://yourendpoint.com`
